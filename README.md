# tor-in-termux
~ proxy for telegram and twitter~
{socks5 and http}

# Download
download termux : termux.com
 
# install
copy code and paste in termux:

```bash
apt update && apt install git -y && cd ~ && git clone https://github.com/samsesh/tor-in-termux.git && bash ~/tor-in-termux/run.sh
```
